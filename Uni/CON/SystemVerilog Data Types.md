## `logic`
- a 4-state data type
- can be driven with both procedural blocks and continuous `assign` statements
- However, a signal with more than one driver needs to be declared a net-type such as `wire`
### 4-state data types
- Types that have 4 kind of values:
1. **unknown - X**
2. **high-impedance - Z**
3. **0**
4. **1**
